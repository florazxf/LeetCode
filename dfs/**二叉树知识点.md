# 基本操作
1. 创建值为v的节点 `TreeNode* node = new TreeNode(v)`
2. 让S是T的左节点 `T->left=S;`  

# 二叉树的知识

1. 一个节点的**最大贡献值**，就是以该节点为根节点的子树中寻找以该节点为起点的一条路径，使得该路径上的节点值之和最大
2. 一个结点的**最大路径和**，就是该节点的值与该节点左右子节点的最大贡献值
